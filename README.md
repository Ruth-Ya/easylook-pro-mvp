# EasyLook Pro – MVP

Mini MVP front de démonstration pour EasyLook Pro.

- Stack : React + Vite
- Upload d'image côté client
- Simulation de traitement "version studio"
- Optimisé mobile (WhatsApp, Instagram, e-commerce)

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
