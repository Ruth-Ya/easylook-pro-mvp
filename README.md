# EasyLook Pro – MVP

Tes photos, version studio.

Une webapp mobile-first pour transformer des photos produits en visuels
qualité studio en moins de 60 secondes, avec paiement sans frais via Mobile Money.

## Démarrage en local

```bash
npm install
npm run dev
```

Ensuite ouvre l'URL indiquée par Vite (généralement http://localhost:5173).

## Déploiement

1. Créer un nouveau repo GitHub (ex: `easylook-pro-mvp`).
2. Pousser ce dossier sur GitHub.
3. Sur Vercel (plan gratuit), cliquer sur **New Project** puis importer le repo.
4. Garder les réglages par défaut (framework : Vite).
